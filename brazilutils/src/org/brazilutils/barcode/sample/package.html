This package contains a sample Swing application for those who wants to play with available barcode symbologies and also for preliminary visual tests for barcode renderers. The application allows you to:

  • View the barcode rendered on the screen;
  • Save the barcode as an image (png or jpeg format);
  • Print the barcode, so you can do read tests using barcode scanners.

To run the sample application, you have to download the BrazilUtils jar file (probably the full distribuction jar or the barcode packages only, that is brazilutils.jar or brazilutils-barcode.jar). In either case do the following:

   $ java -cp brazilutils.jar org.brazilutils.barcode.sample.SwingSample

NOTE For now, you can get the sources from the CVS or download the barcode package from the Documents & Files. To do that go to BrazilUtils home.
